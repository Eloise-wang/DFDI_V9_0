# 二阶段控制逻辑详细说明

本文档详细描述了液压系统控制逻辑中“二阶段”的实现过程、控制逻辑以及相关可调参数。

## 1. 进入二阶段条件
系统从一阶段进入二阶段是**不可逆**的。
- **进入条件**：滤波后的罐压（`display_lng_pressure`，即代码中的滤波压力值） $\ge$ **30.0 MPa**。
- **动作**：系统主状态切换为 `SYSTEM_STATE_REV_PHASE2`，记录进入时间，并根据当前模式（手动/自动）初始化控制参数。

---

## 2. 二阶段模式逻辑

### 2.1 手动模式 (Manual Mode)
当上位机设置 `set_second_manual` 为 **1** 时进入手动模式。
- **控制逻辑**：
  - **换向阀**：严格按照用户在上位机输入的参数运行：
    - 开启（做功）时间 = `set_second_workDone_time` (秒)
    - 关闭（吸油）时间 = `set_second_oilSuction_time` (秒)
  - **旁通阀**：直接跟随用户输入的 `set_bypass_ratio` (0-50%)。
- **适用场景**：调试、应急处理或特定工况下的定频运行。

### 2.2 自动模式 (Auto Mode)
当上位机设置 `set_second_manual` 为 **0** 时进入自动模式。系统根据罐压实时闭环调节。
- **目标区间**：罐压维持在 **30 ± 1.5 MPa** (即 28.5 ~ 31.5 MPa)。
- **调节过程**：
  1. **稳定期**：当罐压在 28.5 ~ 31.5 MPa 之间时，系统保持当前换向频率和旁通阀开度不变。
  2. **欠压调节**：当罐压 **< 28.5 MPa** 时，系统每隔 **1秒** 执行一次增强动作：
     - **增加动力**：旁通阀开度减小 **2%**（`V_new = V_current - 2%`），最低减至 0%。
     - **提高频率**：缩短换向阀的开启和关闭时间，步长为 **100ms**（`T_new = T_current - 0.1s`），最低限制为 0.1s。
  3. **超压调节**：当罐压 **> 31.5 MPa** 时，系统每隔 **1秒** 执行一次减弱动作：
     - **一般超压**（`31.5 < P ≤ 33.0 MPa`）：
       - 旁通阀开度增大 **5%**（`V_new = V_current + 5%`），最大增至 50%；
       - 换向阀开启/关闭时间各增加 **100ms**（`T_new = T_current + 0.1s`），最大限制为 10.0s。
     - **危险超压**（`P > 33.0 MPa`）：
       - 旁通阀开度强制拉到 **50%**（全旁通，停止做功）；
       - 若旁通阀已到 50%，换向阀停止换向并保持在 **OFF** 位，直至压力回落到 `≤ 31.5 MPa` 再恢复正常切换。
- **参数屏蔽**：在自动模式运行期间，系统会忽略用户输入的 `set_second_workDone_time` 和 `set_second_oilSuction_time`，直到切换回手动模式。

---

## 3. 模式切换逻辑
- **手动 $\rightarrow$ 自动**：
  - 系统捕捉切换瞬间，将当前的手动设置时间作为自动模式调节的**初始参考值**，开始闭环计算。
- **自动 $\rightarrow$ 手动**：
  - 系统立即停止闭环算法，直接跳回使用上位机输入的 `set_second_workDone_time` 和 `set_second_oilSuction_time`。

---

## 4. 可调参数汇总

| 参数名称 | 单位 | 描述 | 适用模式 |
| :--- | :--- | :--- | :--- |
| `set_second_manual` | 布尔 | 0: 自动模式, 1: 手动模式 | 全模式 |
| `set_second_workDone_time` | 秒 | 换向阀开启（做功）持续时间 | 手动模式 |
| `set_second_oilSuction_time` | 秒 | 换向阀关闭（吸油）持续时间 | 手动模式 |
| `set_bypass_ratio` | % | 旁通阀目标开度 | 手动模式 |
| `set_first_fix_freq_time_on` | 秒 | 一阶段开启时间（作为自动模式初值） | 自动模式起始 |
| `set_first_fix_freq_time_off` | 秒 | 一阶段关闭时间（作为自动模式初值） | 自动模式起始 |

## 5. 二阶段控制常量（已宏化，便于调整）
这些参数已在 [hydraulic_control.h](file:///d:/DOWNLOAD/DFDI_V9_0/Inc/App/hydraulic_control.h) 中以宏集中定义（`PHASE2_*`），二阶段自动模式逻辑直接引用这些宏。
- **压力目标值**：30.0 MPa（`PHASE2_TARGET_PRESSURE_MPA`）
- **容差范围**：± 1.5 MPa（`PHASE2_PRESSURE_TOLERANCE_MPA`，对应上下限 `PHASE2_PRESSURE_LOW_MPA / PHASE2_PRESSURE_HIGH_MPA`）
- **危险超压阈值**：33.0 MPa（`PHASE2_PRESSURE_DANGER_MPA`）
- **调节周期**：1000 ms (1秒)（`PHASE2_ADJUST_PERIOD_MS`）
- **旁通阀调节步长**：2%（`PHASE2_BYPASS_STEP_PERCENT`）
- **旁通阀超压步长**：5%（`PHASE2_BYPASS_OVER_STEP_PERCENT`）
- **时间调节步长**：100 ms (0.1秒)（`PHASE2_TIME_STEP_S`）
- **最小换向时间限制**：0.1 秒（`PHASE2_MIN_VALVE_TIME_S`）
- **最大换向时间限制**：10.0 秒（`PHASE2_MAX_VALVE_TIME_S`）

## 6. 换向频率统计（滑动窗口法）
换向频率用于上位机显示与监控，统计口径为“过去 60 秒内的总换向次数”（次/分钟）。
- **统计口径**：一次“换向”= 两次物理翻转（ON->OFF->ON 或 OFF->ON->OFF）。
- **统计实现**：1 秒分辨率的 60 秒滑动窗口（环形队列 `counts_per_sec[60]`）。
- **刷新周期**：每秒更新一次窗口并计算总和输出 `rev_freq_per_min`。
- **抖动处理**：若调度抖动导致更新间隔超过 1 秒，会推进多个桶，缺失的秒填 0，避免窗口滞后导致显示不可信。
- **上位机范围**：上传 CAN 的 `rev_freq` 信号受 DBC 限制为 0..255；超过 255 的内部值会被截断到 255。
