# 二阶段控制逻辑详细说明

本文档详细描述了液压系统控制逻辑中“二阶段”的实现过程、控制逻辑以及相关可调参数。

## 1. 进入二阶段条件
系统从一阶段进入二阶段是**不可逆**的。
- **进入条件**：在一阶段运行过程中检测到油压（`oil_pressure`）$\ge$ 设定的伸出压力 `set_extend_pressure`。
- **动作**：系统主状态切换为 `SYSTEM_STATE_REV_PHASE2`，记录进入时间，并将二阶段循环状态初始化为“回缩定时”作为首动作。

---

## 2. 二阶段模式逻辑

### 2.1 手动模式 (Manual Mode)
当前二阶段实现基于“手动模式”流程：通过“压力阈值 + 定时动作”的组合实现伸出/回缩循环。
- **输入参数（来自 tsmaster_control2）**：
  - `set_extend_pressure`：伸出压力阈值（MPa）
  - `set_extend_time`：伸出动作保持时间（秒）
  - `set_retract_pressure`：回缩压力阈值（MPa）
  - `set_retract_time`：回缩动作保持时间（秒）
- **控制逻辑（循环执行）**：
  1. **等待高阈值**：保持换向阀在“伸出/做功”（ON）位，等待油压 `oil_pressure >= set_extend_pressure`。
  2. **回缩定时**：触发回缩（OFF），并保持 `set_retract_time` 秒。
  3. **等待低阈值**：保持回缩（OFF）位，等待油压 `oil_pressure <= set_retract_pressure`。
  4. **伸出定时**：触发伸出（ON），并保持 `set_extend_time` 秒。
  5. 回到步骤 1。
- **滞回区间**：`set_extend_pressure` 与 `set_retract_pressure` 构成滞回，避免油压在阈值附近波动导致阀频繁来回动作。

### 2.2 自动模式 (Auto Mode)
当上位机设置 `auto_mode` 为 **1** 时进入自动模式（预留接口）。
- **当前状态**：自动模式的闭环/自适应算法尚未实现；`auto_mode` 仅用于对齐上位机信号并预留后续扩展。
- **当前行为**：即使 `auto_mode==1`，二阶段仍按 2.1 的“压力阈值 + 定时动作”流程执行。

---

## 3. 模式切换逻辑
- **手动 $\rightarrow$ 自动**：当前仅改变标志位，控制流程不变（仍按 2.1 执行）。
- **自动 $\rightarrow$ 手动**：当前仅改变标志位，控制流程不变（仍按 2.1 执行）。

---

## 4. 可调参数汇总

| 参数名称 | 单位 | 描述 | 适用模式 |
| :--- | :--- | :--- | :--- |
| `auto_mode` | 布尔 | 0: 手动模式, 1: 自动模式（预留） | 全模式 |
| `set_extend_pressure` | MPa | 伸出压力阈值（达到后触发回缩定时） | 二阶段 |
| `set_retract_pressure` | MPa | 回缩压力阈值（低于后触发伸出定时） | 二阶段 |
| `set_extend_time` | 秒 | 伸出动作保持时间 | 二阶段 |
| `set_retract_time` | 秒 | 回缩动作保持时间 | 二阶段 |
| `set_bypass_ratio` | % | 旁通阀目标开度 | 手动模式 |
| `set_first_fix_freq_time_on` | 秒 | 一阶段开启时间（作为自动模式初值） | 自动模式起始 |
| `set_first_fix_freq_time_off` | 秒 | 一阶段关闭时间（作为自动模式初值） | 自动模式起始 |

## 5. 换向频率统计（滑动窗口法）
换向频率用于上位机显示与监控，统计口径为“过去 60 秒内的总换向次数”（次/分钟）。
- **统计口径**：一次“换向”= 两次物理翻转（ON->OFF->ON 或 OFF->ON->OFF）。
- **统计实现**：1 秒分辨率的 60 秒滑动窗口（环形队列 `counts_per_sec[60]`）。
- **刷新周期**：每秒更新一次窗口并计算总和输出 `rev_freq_per_min`。
- **抖动处理**：若调度抖动导致更新间隔超过 1 秒，会推进多个桶，缺失的秒填 0，避免窗口滞后导致显示不可信。
- **上位机范围**：上传 CAN 的 `rev_freq` 信号受 DBC 限制为 0..255；超过 255 的内部值会被截断到 255。
